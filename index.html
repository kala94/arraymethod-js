Method Name Process Changes of the array Return
push() Adds the specified elements to the end of an array. It change the original array. Returns the new length of the array.
pop() Removes the last element from an array. It change the original array Returns the remove element.
shift() Removes the first element of the array. It change the original array Returns the remove element.
unshift() Add the first element of the array. It change the original array. Returns the new length of the array.
length() The number of elements in that array. Does not change the original array. Returns the numeric value.
join() Creates and returns a new string by concatenating all of the elements in this array. Does not change the original array. Empty string is returned.
tostring() Returns a string representing the specified array and its elements. Does not change the original array. Return a string value.
concat() Adds the new string to the old string and returns the overall string. Does not change the existing arrays. Returns a new string.
splice() Changes the contents of an array by removing or replacing existing elements and/or adding new elements in place. Removing, replacing, or adding values and returns the affected values. If only one element is removed, an array of one element is returned. If no elements are removed, an empty array is returned.
slice() Returns a shallow copy of a portion of an array into a new array object selected from start to end ( end not included) where start and end represent the index of items in that array. The original array will not be modified. Returns selected elements in an array, as a new array.
indexof() Starts at a specified index and searches from left to right. Does not change the original array. Returns the first index at which a given element can be found in the array, or -1 if it is not present.
lastindexof() String values searches this string and returns the index of the last occurrence of the specified substring. Does not change the original array. Returns the index from the beginning (position 0). The lastIndexOf() method returns -1 if the value is not found.
